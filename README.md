### Description

Warm-weather is a website that helps you decide what to wear based on the weather conditions. The web app pull weather condition base on client GPS location or base on client zip code.

---

### Tech Used

#### Frontend

- React
- Tailwind
- weatherAPI.com

#### Backend

- Django
- restful famework
- openai api
